TCP BBR 拥塞控制快速部署指南

发表于 2017-04-22 更新于 2021-12-17

这个由 Google 提交到 Linux Kernel 主线的 TCP BBR (Bottleneck Bandwidth and RTT) 拥塞控制算法致力于解决两个问题：

  • 在有一定丢包率的网络链路上充分利用带宽
  • 降低网络链路上的 buffer 占用率，从而降低延迟

部署 TCP BBR 的先决条件：

  • 需要使用 linux 4.9.0-rc8 或更高版本的内核
